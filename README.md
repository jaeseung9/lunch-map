# 🍽️ 식권대장 가맹점 지도

동대문 캠퍼스 주변 식권대장 사용 가능 가맹점을 지도에서 쉽게 찾을 수 있는 웹 애플리케이션

## 🔗 링크

**웹사이트:** https://jaeseung9.github.io/lunch-map/

## 📌 주요 기능

- 카카오맵 기반 가맹점 위치 표시
- 가맹점 검색 기능
- 마커 클릭 시 상세 정보 표시
- 가맹점 리스트 클릭으로 지도 이동
- 반응형 디자인 (모바일 지원)
